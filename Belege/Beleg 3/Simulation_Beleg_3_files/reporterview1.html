  • Ist eine Zelle tot (Matrixeintrag Null), so wird sie in der Folgegeneration lebendig (Matrixeintrag 1), wenn genau drei ihrer Nachbarzellen in der aktuellen Generation leben.
  • Ist eine Zelle lebendig, so stirbt sie in der Folgegeneration, wenn sie weniger als zwei lebende Nachbarzellen hat (Einsamkeit) oder mehr als drei lebende Nachbarzellen (Überbevölkerung).
  • Ist eine Zelle lebendig, so bleibt sie in der Folgegeneration am Leben, wenn sie aktuell genau zwei oder drei lebende Nachbarzellen hat.
